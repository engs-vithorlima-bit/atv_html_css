HTML

Linguagem de marcação para criar páginas web. Estrutura o conteúdo (o “esqueleto” da página). Define elementos como:
  • Títulos (h1, h2)
  • Parágrafos (p)
  • Imagens (img)
  • Links (a)
  • Listas (ul, ol) Exemplo: h1 ,Meu site h1 → cria um título grande com o texto “Meu site”.

    CSS

    Linguagem de estilo para páginas web.
  • Controla a aparência do conteúdo (cores, fontes, tamanhos, layouts).
  • Permite personalizar elementos HTML:
  • Cor do texto (color)
  • Tamanho da fonte (font-size)
  • Margens e espaçamento (margin, padding)
  • Alinhamento e posição (text-align, position) Exemplo: h1 { color: blue; font-size: 36px; } → deixa o título azul e maior Google
